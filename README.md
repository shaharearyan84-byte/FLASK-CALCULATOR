## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/flask-calculator.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd flask-calculator
```

### 3️⃣ Install Flask

```bash
pip install flask
```

### 4️⃣ Run the Application

```bash
python app.py
```

### 5️⃣ Open in Browser

```text
http://127.0.0.1:5000
```
